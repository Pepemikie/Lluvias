# Athmosphere

Programa en C que te dice cual es el evento astronomico mas proximo, ya sea luna llena/nueva y/o lluvia de estrellas visible (que tambien variará segun la luna (porq si es llena, las estrellas se ven menos)), con un Arduino mostrando el resultado en LED/LCD:

Planes a futuro, que guarde el estado del dia anterior por si no se ha sincroniado bien marcar en el LCD un no sync; que el leer cuando toca la luna nueva y llena se vea a través de un calendario lunar que sea coherente. QUe las lluvias de estrellas sean actualizadas a traves de una base de datos fiable. que cuente cuantos dias falta para el evento. y separar por botones evento lunar y evento estelar
